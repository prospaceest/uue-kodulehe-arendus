# BRAND_VOICE_RU.md — Russian voice rules for Varjuprofiilid.ee

These are the **Russian equivalents** of the 9 brand voice rules in `CLAUDE.md`. Same intent: factual, restrained, no marketing puffery. Translator MUST follow these — Google Translate-tier output is not acceptable.

---

## 1. No "shadow gap" jargon
**EST:** Use *varjujoon* / *varjuefekt*, never anglicism.
**RU:** Use *теневой шов* (most natural) or describe the effect: *тонкая теневая линия*, *скрытая линия*. **Never write «shadow gap»** in Russian copy. Avoid the Anglicism even when it's tempting.

## 2. No "subtiilne"
**EST:** Use *peen* / *vaoshoitud*.
**RU:** Use *тонкий* (slim), *сдержанный* (restrained), *деликатный* (sparingly). **Never «изысканный»**, **«утончённый»**, or **«элегантный»** — they read as marketing fluff.

## 3. No "premium" or marketing puffery
**RU:** **Never write «премиум»**, **«люкс»**, **«элитный»**, **«роскошный»**. State facts. The product is good; that's enough.

## 4. No "floating furniture effect"
**EST:** *Hõljuva mööbli efekt* is wrong (furniture doesn't float). *Hõljuva lae / hõljuva seina* are OK.
**RU:** Don't write *парящая мебель* / *эффект парящей мебели*. *Парящий потолок* / *парящая стена* are OK — describe the wall/ceiling, not the furniture.

## 5. "Below the drywall", not "behind it"
**EST:** *Kipsplaadi alla*, not *kipsplaadi taha*.
**RU:** *Под гипсокартоном* — the profile sits under the drywall plane. **Never «за гипсокартоном»** — that's structurally wrong (would mean inside the cavity).

## 6. Color names ≠ process
**EST:** "Anodeeritud hõbe" is a *color name* matching the anodized look — not the anodizing process.
**RU:** *«Анодированное серебро»* should be the **color name** — like "silver chrome" or "anodized silver finish". Don't claim profiles ARE anodized unless they are. If translator is unsure, write *цвет «серебро»* and add note for review.

## 7. LED specs by category — write Cyrillic, not Latin/mixed
- **Ceiling profiles:** `12 Вт/м · 234 LED/м · 3000K · 24В`
- **Floor / wall profiles:** `COB 16 Вт/м · 1350 лм/м · CRI 94 · 24В`

Notes:
- Use `Вт` not `W`, `м` not `m`, `В` not `V`. (Russian convention.)
- `LED` stays Latin — common in Russian tech writing.
- `COB` stays Latin — technical abbreviation.
- `K` (Kelvin) stays Latin — international unit.
- `CRI` stays Latin — color rendering index, standard.

## 8. Standard lengths
- Ceiling profiles: **2,5 м** (AST35 = 3 м)
- Floor profiles: **2,6 м**
- LPA909 / LPA126: **3 м**

Use Russian comma decimal (2,5 м not 2.5m).

## 9. VAT — 24%
**RU:** *с НДС 24%* — every price already includes Estonian VAT (since July 2025).
Wording on product pages: *«цена за 1 пог.м, с НДС 24%»* (per linear meter, including VAT).
**Currency:** Always €. Never write ₽ or RUB — this is Estonia.

## What NOT to do in Russian copy

- **No transliterated Estonian.** Never write «сисекуйундая», «варьюпрофииль», «вальгусриба» etc. Translate concepts, don't transliterate.
- **No machine-translation tics.** Telltale signs: literal word order from EST, missing aspectual verbs (perfective/imperfective), forced relative clauses. A native Russian speaker should pass a sentence and not feel that it was generated.
- **No emoji** in product copy. Lightning bolt `⚡` for LED is OK (matches EST convention) — that's it.
- **No mixed scripts mid-sentence** unless technical (e.g. *RAL 9005* — keeps Latin; *24 В* — uses Russian). LED, COB, CRI, K stay Latin. Everything else Cyrillic.
- **No imperative sales tone.** Russian e-commerce often slips into *«Купите сейчас!»* / *«Не упустите!»* — we don't do that.
- **No Eastern Slavic regionalisms.** Some EST→RU translators inadvertently use Belarusian/Ukrainian colloquialisms. Use **standard Russian** as taught/written in Russia or Estonian-Russian press.

## Estonian-Russian context

Our audience is **Russian-speakers living in Estonia** — not Russia, not Belarus.

- They read Estonian-Russian press: Postimees Russian edition, Delfi RU, ERR RU.
- They use Google.ee (in Russian) + Yandex (~15–20% of them).
- Many are bilingual; mixed-script searches like *"AST22 цена"* or *"LED-profiil Tallinnas"* are normal.
- Cultural references: Tallinn, Tartu, Pärnu, Lasnamäe, Mustamäe (transliterate Estonian place-names: Ласнамяэ, Мустамяэ).
- **Don't reference Russia-specific brands, retailers, or media** ("М.Видео", "ВКонтакте" — except where genuinely useful). Default audience reference points are Estonian.

## Translator handoff

When a professional EST→RU translator picks up this project:

1. **Read this file in full** before touching content.
2. **Reference document** for terminology: `CLAUDE.md` (EST rules) — many concepts are 1:1 mappings.
3. **Don't change the URL slugs** (`urlPathRu` fields are SEO-tuned, don't redo them).
4. **Long descriptions** follow the same EST template structure: 134–155 words, what + how + finish + warranty + price note. Don't shorten.
5. **Specs** (LED specs, lengths, colors) are factual constants — verify, don't paraphrase.
6. **Sign your work**: add `* translator: <name>` line at the top of any file you edit so we know what was professionally reviewed.

When in doubt, **ask** rather than guess. Some Estonian-specific concepts (Vana-Kalamaja street, EE reg numbers, Eesti Energia conventions) need contextual transliteration, not creative rendering.
